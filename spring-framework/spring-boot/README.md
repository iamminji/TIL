# 스프링 부트

## Servlet Filter
`filter` 는 어플리케이션의 HTTP request, response 를 중간에 가로채어 사용할 수 있는 객체이다.
`filter` 를 사용하면 아래 두 기능을 수행할 수 있다.

- request가 controller에게 전송되기 전
- response가 client에게 전송되기 전
