
unit15. if 조건문 사용하기
==
  
  
작성할 때 __()__ 괄호를 사용하지 않고 조건문을 시작하는 줄에서 여는 중괄호 __{__ 가 시작된다. 즉,

<pre><code>i := 10

if i >= 5 // 컴파일 에러!!
{
    fmt.Println("5 이상")
}
</code></pre>
  
위와 같이 중괄호가 다음 줄에서 시작하면 컴파일 에러가 난다. 생략도 안된다. <code>else</code>나 <code>else if</code>도 마찬가지다. 
근데 <code>else</code> 다음에 오는 여는 괄호는 밑에 있어두 괜찮더라.
