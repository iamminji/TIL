
스트림 (Stream)
==============

__스트림__ 이란 데이터 처리 연산을 지원하도록 소스에서 추출된 연속된 요소로 정의할 수 있다.


스트림은 자바 API에 새로 추가된 기능으로, 스트림을 이용하면 선언형(즉, 데이터를 처리하는 임시 구현 코드 대신 질의로 표현할 수 있다.) 으로 
컬렉션 데이터를 처리할 수 있다.

          apple
          orange
          coke
          ============Java8=============
          apple
          orange
          coke
</code></pre>
