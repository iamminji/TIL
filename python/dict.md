# Dict

파이썬 3.6 이상 부터는 기본 `dict` 가 순서가 보장된다고 한다

Python2.7
```
>>> a = {"1": 2}
>>> a["2"] = 3
>>> a["4"] = 5
>>> a["6"] = 5
>>> a
{'1': 2, '2': 3, '4': 5, '6': 5}
>>> a["3"] = 5
>>> a
{'1': 2, '3': 5, '2': 3, '4': 5, '6': 5}
```

Python3.7
```
>>> a = {"1": 2}
>>> a["2"] = 3
>>> a["4"] = 5
>>> a["6"] = 5
>>> a
{'1': 2, '2': 3, '4': 5, '6': 5}
>>> a["3"] = 5
>>> a
{'1': 2, '2': 3, '4': 5, '6': 5, '3': 5}
```
