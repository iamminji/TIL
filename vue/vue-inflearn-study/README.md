### 뷰 컴포넌트
화면의 영역을 구분해서 코드로 관리 하는 것이 컴포넌트이다.

#### 전역 컴포넌트와 지역 컴포넌트의 차이점
전역 컴포넌트는 플러그인, 라이브러리에서만 주로 사용한다.
일반적으로는 지역 컴포넌트로 등록해 나가면 된다.
